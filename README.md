# AI Support + Sales Copilot

A sellable RAG-based AI chatbot for startup companies.

## Goal
Build an AI chatbot that:
- answers support questions from company data
- captures and qualifies leads
- supports startup customer operations
- can be integrated into company websites and systems

## Stack
- Backend: FastAPI
- Frontend: Node.js / Next.js
- Infra: Docker
- CI/CD: GitHub Actions
